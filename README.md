# Apollo Flooring CRM
### Setup & Usage Guide

---

## How to Open the App

1. **Download** the `apollo-flooring-crm` folder to your computer
2. **Double-click** `index.html` — it opens in your web browser
3. That's it. No internet required, no installation needed.

---

## Where Your Data Is Saved

All customer data, orders, notes, and price sheets are saved automatically in your **browser's local storage** on that computer. This means:

- ✅ Data persists when you close and reopen the browser
- ✅ Works completely offline
- ⚠️ Data is tied to the **browser + computer** you use (e.g. Chrome on the front desk PC)
- ⚠️ Data does **not** automatically sync to other computers or phones yet

**Recommendation:** Keep the app open on one dedicated computer (e.g. the front desk PC). When Microsoft 365 is connected, all data can move to the cloud automatically.

---

## Best Setup for Your Team Right Now

1. Put the `apollo-flooring-crm` folder on a **shared drive or USB** so everyone has a copy
2. Each salesman opens it on **their own device** for the walk-in log
3. At the end of the day, use **Export Data** (see below) to back up
4. The office can run it on the front desk PC as the "master" copy

---

## Backing Up Your Data

To back up or transfer data between computers:

1. Open the app in Chrome or Edge
2. Press **F12** (Developer Tools) → **Application** tab → **Local Storage** → your file
3. Copy the value and paste into a text file — this is your backup

*(We can add an Export button to make this easier — just ask!)*
